# Roadmap

## Now

- Keep the CLI dependency-free and easy to audit.
- Improve filename and path-based review signals.
- Collect false positive reports from real pull requests.

## Next

- Add repository-local configuration for custom review categories.
- Add optional JSON schema for stable automation output.
- Add richer GitHub Action examples for job summaries and artifacts.

## Later

- Prototype Codex-assisted diff summaries.
- Suggest missing tests or docs from changed hunks.
- Draft editable maintainer review comments.
- Explore optional security-focused adapters while keeping the default CLI conservative.
